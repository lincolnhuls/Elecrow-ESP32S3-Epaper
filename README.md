All files in this folder must be included in the upload when using this code. 
This code only works for the 3.7 inch display.

Follow upload scheme as follows: 
  Flash settings for this board are fairly specific
  
  Board: ESP32S3 Dev Module 
  Flash Size: 8MB (64Mb)
  Partition Scheme: "Huge APP (3MB No OTA/1MB SPIFFS)"
  PSRAM: "OPI PSRAM"
  Upload Speed: "460800" (Board will not upload if this is not set)
